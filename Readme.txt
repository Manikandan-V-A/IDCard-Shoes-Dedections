

To run:

1) Download Git bash : https://gitforwindows.org/

2) Open Git bash with source code path.

	2.1) Type for initiate: export PYTHONIOENCODING=utf-8 or set PYTHONIOENCODING=utf-8

3) Install packages: pip install -r ./requirements1.txt

4) Type: chmod +x ./app.sh

5) Place the image to detect

6) Run shell file: ./app.sh img_name.jpg

NOTE: Do not use anaconda prompt for it. Use python version 3.9
